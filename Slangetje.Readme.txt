Snake Game – Webversie
 Over dit project

Dit project is een browser-gebaseerde Snake Game gemaakt met HTML5 Canvas, CSS en JavaScript.
De speler bestuurt een slang die burgers eet om punten te verzamelen. Tijdens het spel verschijnen bommen die levens kosten wanneer je ze raakt.

Het doel van het spel is om een zo hoog mogelijke score te behalen zonder tegen muren, jezelf of te veel bommen aan te botsen.

Gameplay--

De speler kan:

bewegen met WASD
bewegen met pijltjestoetsen
burgers eten om punten te verdienen
bommen vermijden om levens te behouden
sneller spelen naarmate de score stijgt

De speler start met:

❤️ 3 levens
🍔 score = 0
💣 bom verschijnt vanaf score 3

Wanneer alle levens op zijn is het spel afgelopen.

⚙️ Gebruikte technieken

Dit project is gemaakt met:

HTML5
CSS3
JavaScript
Canvas API

Waarom deze technieken?

HTML en Canvas zorgen voor het speelveld
CSS maakt de interface duidelijk en modern
JavaScript regelt de game-logica en interactie

Deze combinatie werkt goed voor browsergames zonder extra frameworks.
